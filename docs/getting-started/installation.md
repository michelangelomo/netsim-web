# Installation

NetSimWeb runs entirely in your browser - no installation required!

## Using the Hosted Version

Simply visit [NetSimWeb App](../app/) to start using the simulator.

## Running Locally

If you want to run NetSimWeb locally for development:

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/netsimweb.git
   cd netsimweb
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
```

The static output will be in the `out/` directory.
