# Creating Connections

Connect your network devices to build your topology.

## Making Connections

### Method 1: Interface Selection

1. Click on the first device
2. Click on an available interface port
3. Click on the second device
4. Select the target interface

### Method 2: Cable Tool

1. Select the cable tool from the toolbar
2. Click on the source device
3. Drag to the destination device
4. Release to create the connection

## Connection Types

Connections are automatically created based on device types:

| Connection | Use Case |
|------------|----------|
| Ethernet | Standard device connections |
| Serial | Router-to-router WAN links |
| Console | Management connections |

## Managing Connections

### Delete a Connection

1. Click on the connection line
2. Press `Delete` or use the delete button

### View Connection Details

Click on a connection to see:

- Connected interfaces
- Link speed
- Status
