# Interface Overview

NetSimWeb has a clean, intuitive interface designed for ease of use.

## Main Components

### Sidebar

The left sidebar contains:

- **Device Palette** - Drag devices onto the canvas
- **Statistics** - View network stats at a glance

### Canvas

The central canvas is your workspace where you:

- Place and arrange network devices
- Create connections between devices
- View packet animations during simulation
- Use the top-right controls to zoom, fit view, or **auto-arrange** devices (Grid icon) into a readable layout

### Properties Panel

When you select a device, the properties panel shows:

- Device name and type
- Interface configurations
- IP addresses and status

### Terminal

Each device has an interactive terminal for CLI configuration:

- Type commands directly
- View command output
- Configure interfaces and routing

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Delete` | Remove selected device |
| `Escape` | Deselect / Cancel |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
